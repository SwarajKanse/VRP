This is the VRP Platform - a complete vehicle routing and logistics operations
system built with pure Python.

QUICK START:
  1. Run setup.bat (Windows) or setup.sh (Linux/macOS)
  2. Verify with: python test_installation.py
  3. Launch with: python -m vrp_platform.ui.app
  4. Open http://localhost:8080

PRIMARY DOCUMENTATION:
  - README.md           Complete platform overview
  - QUICKSTART.md       Step-by-step setup guide
  - GETTING_STARTED.txt Detailed getting started guide

TESTING:
  - python -m pytest tests/platform -q

REQUIREMENTS:
  - Python 3.11 or higher
  - No compilation needed
  - No C++ dependencies
  - Works on Windows, Linux, and macOS

WHAT'S INCLUDED:
  - Multi-workspace web application (NiceGUI)
  - Pure-Python routing optimizer
  - SQLAlchemy persistence layer
  - Dispatcher control tower
  - Warehouse operations
  - Driver workflow
  - Customer portal
  - Admin dashboard
